OpenFin Not Loaded!

This application should launch the OpenFinCenter. Use the buttons above to send notifications. Minimize the OpenFinCenter via tray icon to see small notifications. If OpenFinCener is open, the notifications will appear in the application.

Notification Icon is pulled the from sending applications manifest, the app name is the UUID. Title and Text can be sent with invoking the notification.

To invoke notifications use 'new fin.desktop.Notification.Notification2("Notification Title", "Notification Text"); '' Preload script required. See /preload/fin.js.Notification.Notification2.js
